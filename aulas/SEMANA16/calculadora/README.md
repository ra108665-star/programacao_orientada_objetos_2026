# Calculadora Swing

O projeto requer um JDK instalado. Para compilar e executar pelo terminal, entre
nesta pasta e use:

```sh
javac -d bin $(find src -name "*.java")
java -cp bin br.com.exemplo.calculadora.Main
```

No VS Code, também é possível abrir
`src/br/com/exemplo/calculadora/Main.java` e usar **Run Java**.
