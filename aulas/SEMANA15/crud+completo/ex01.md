# Exercício 1 – Entender o projeto atual

## 1. Qual classe representa uma entidade?

`Produto` e `Orcamento`.

## 2. Qual classe recebe requisições HTTP?

`ProdutoController` e `OrcamentoController`.

## 3. Qual classe possui regras de negócio?

`ProdutoService` e `OrcamentoService`.

## 4. Qual interface conversa com o banco?

`ProdutoRepository` e `OrcamentoRepository`

## 5. Qual endpoint cria um orçamento?

O endpoint responsável por criar um orçamento é:

```http
POST /orcamentos
```
Esse endpoint é definido em `OrcamentoController` e chama o `OrcamentoService` para realizar a operação.

## 6. Qual endpoint lista produtos?

O endpoint responsável por listar todos os produtos é:

```http
GET /produtos
```
Esse endpoint é definido em `ProdutoController` e retorna a lista de produtos cadastrados no banco de dados.
